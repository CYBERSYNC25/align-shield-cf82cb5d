# Roteiro de Teste de Usabilidade - Compliance Sync

## 📋 Objetivo do Teste
Avaliar a clareza da navegação, facilidade de uso das interfaces principais e efetividade do novo sistema de onboarding contextual.

## 🎯 Cenário de Uso
Você é um profissional responsável por gerenciar compliance na sua organização. Foi designado para avaliar uma nova ferramenta de gestão de compliance e precisa entender como ela funciona.

### Tarefa 3: Conectar uma Integração
**Objetivo**: Avaliar clareza do processo de conexão

**Instruções**:
1. Na aba "Catálogo", localize a integração "Google Workspace"
2. Clique no botão para conectar (não precisa completar o processo real)
3. Observe as informações apresentadas no modal
4. Feche o modal e explore as estatísticas no topo da página

**Perguntas de Feedback**:

- [ ] O botão de conexão é claro? (1-5) ___
- [ ] As informações sobre a integração são suficientes? (1-5) ___

## ✅ Checklist do Facilitador

Antes do teste:
- [ ] Ambiente configurado e funcionando
- [ ] Dados de exemplo carregados
- [ ] Cronômetro preparado
- [ ] Formulário impresso ou digital pronto

Durante o teste:
- [ ] Observar expressões faciais e hesitações
- [ ] Encorajar o usuário a "pensar alto"
- [ ] Não intervir, apenas observar (exceto se travado)
- [ ] Anotar tempo em cada tarefa

Após o teste:
- [ ] Revisar anotações
- [ ] Consolidar feedback
- [ ] Priorizar problemas críticos
- [ ] Compartilhar com equipe de desenvolvimento
